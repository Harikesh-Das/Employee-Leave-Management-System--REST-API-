
Key Features

- Authentication & Authorization: Secure JWT-based authentication with role-based access control (RBAC).
- Employee Management: Full CRUD operations for employee records.
- Leave Management: APIs to apply for, cancel, approve, reject, and track leave requests.
- Dashboard Analytics: Aggregated statistics for admin/manager dashboards (employees, total leaves, pending, approved, and rejected requests).

 Tech Stack

- Backend: Node.js, Express.js
- Database: SQLite
- ORM: Sequelize
- Authentication: JWT
